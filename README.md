# 🧠 Capítulo 1: Os Alicerces da Lógica (Algoritmos e Estruturas)

"Antes de construir arranha-céus, é preciso garantir que a base seja 
sólida. Aqui é onde aprendi a falar a língua dos computadores."

Este repositório reúne desafios práticos desenvolvidos em Portugol Studio. 
Como um Full Stack Developer em formação, entendi que a tecnologia muda, 
mas a lógica por trás de uma solução com propósito é universal.

---

## 📖 A HISTÓRIA

Nesta fase da minha jornada, deixei de ser apenas um usuário de tecnologia 
para me tornar uma criadora. Enfrentei o desafio de organizar dados em 
vetores e criar algoritmos que pudessem ordenar o caos. Cada loop e cada 
condicional foi um passo para entender como transformar problemas complexos 
em instruções simples e eficientes.

---

## 💻 DESAFIOS RESOLVIDOS

Neste capítulo, foquei em manipulação de dados e aritmética lógica:

* Ordenação de Vetores: Implementação de algoritmos para organizar números 
  em ordem crescente[cite: 8].
* Processamento de Dados: Leitura de 10 números para extração de métricas 
  relevantes[cite: 4].
* Filtros Inteligentes: Identificação de elementos em índices ímpares e 
  separação de números pares[cite: 4].
* Análise Estatística: Cálculo automatizado de soma e média de valores 
  inseridos[cite: 4].
